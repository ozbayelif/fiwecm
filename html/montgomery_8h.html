typedef struct Montg_Curve_s MONTG_CURVE_t[1]

Structure to represent a Montgomery curve.

Curve Equations:

  • Affine Coordinates: B*y^2 = x^3 + A*x^2 + x
  • Projective Coordinates: B*y^2*Z = x^3 + A*x^2*Z + x*Z^2
